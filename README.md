# Mon portfolio

Ce dépôt contient le code de mon portfolio développeur, qui présente mes projets, mes compétences et mes expériences.

## 🎯 Objectif

- Mettre en avant mes projets personnels et scolaires.
- Montrer les technologies que j’utilise au quotidien.
- Servir de point d’entrée pour les recruteurs et collaborateurs.

## 🖥️ Aperçu

- Page d’accueil avec une courte présentation.
- Section projets avec description, liens et captures d’écran.
- Section compétences (langages, frameworks, outils).
- Liens vers mes profils (GitHub, LinkedIn, etc.).

_Si tu as un déploiement en ligne, ajoute ici :_

- Démo en ligne : [https://ton-portfolio.com](https://ton-portfolio.com)

## 🛠️ Tech stack

- Frontend : React / Vue / Svelte / HTML-CSS-JS (à adapter)
- Style : Tailwind CSS / SCSS / CSS modules (à adapter)
- Outils : Vite / Webpack / Node.js / npm / pnpm / bun (à adapter)

## 🚀 Installation

```bash
# Cloner le dépôt
git clone https://github.com/Gaspami/portfolio.git
cd portfolio

# Installer les dépendances
npm install  # ou pnpm install, yarn, etc.

# Lancer en développement
npm run dev

# Build de production
npm run build
